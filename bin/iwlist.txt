NAME
       iwlist - display detailed wireless interface information

SYNOPSIS
       iwlist interface [scanning | frequency | rate | power | retry | auth | keys | modulation]

DESCRIPTION
       iwlist is a diagnostic tool used to display detailed wireless information
       for IEEE 802.11 interfaces using Linux Wireless Extensions (WEXT).

       This build of iwlist is part of the HC1703 IP camera firmware and is derived
       from the open-source Wireless Tools package by Jean Tourrilhes.

       The code is based on iwlist.c from wireless-tools version 29 (circa 2009),
       with minor backports from version 30, and has been modified for embedded use.

COMMANDS
       scanning
              Perform a scan and list detected access points.

       frequency
              Display available channels and frequencies.

       rate
              Show supported bitrates.

       power
              Display power management capabilities.

       retry
              Show retry limits and lifetime.

       auth
              Show authentication and WPA capabilities.

       keys
              Display encryption key status.

       modulation
              Show supported modulation schemes.

IMPLEMENTATION
       This program uses Linux Wireless Extensions ioctls including:

           SIOCGIWSCAN
           SIOCGIWRATE
           SIOCGIWFREQ
           SIOCGIWPOWER
           SIOCGIWRETRY
           SIOCGIWENCODE
           SIOCGIWAUTH

       Output formatting and data parsing follow the upstream Wireless Tools design.
